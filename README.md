# Baby Princess Ecommerce Website With Django + React


## EN


Original project can be found [here](https://github.com/dennisivy/proshop_django).

# Features
* Full featured shopping cart
* Product reviews and ratings
* Top products carousel
* Product pagination
* Product search feature
* User profile with orders
* Admin product management
* Admin user management
* Admin Order details page
* Mark orders as delivered option
* Checkout process (shipping, payment method, etc)
* Pix / Boleto / Credit card integration

# Download and setup instructions

* Make sure [Python](https://docs.python.org/3/using/index.html) is installed and properly configured on your machine.

### In the terminal, use the following commands (Windows):

* 1 - `git clone https://github.com/Kakamc12/Trabalho-DAS`
* 2 - `cd Trabalho-DAS`
* 3 - Create a virtual environment: `virtualenv myenv`
* 4 - `myenv\scripts\activate`
* 5 - `pip install -r requirements.txt`
* 6 - Install React modules: `cd frontend`
* 7 - `npm install`
* 8 - `python manage.py runserver`
