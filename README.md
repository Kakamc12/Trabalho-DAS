# Download and setup instructions

* Make sure [Python](https://docs.python.org/3/using/index.html) is installed and properly configured on your machine.

### In the terminal, use the following commands (Windows):

* 1 - `git clone https://github.com/Kakamc12/Trabalho-DAS`
* 2 - `cd Trabalho-DAS`
* 3 - Create a virtual environment: `virtualenv myenv`
* 4 - `myenv\scripts\activate`
* 5 - `pip install -r requirements.txt`
* 6 - Install React modules: `cd frontend`
* 7 - `npm install`
* 8 - `python manage.py runserver`


### Administrator 

* 1- Go to the terminal type `python manage.py createsuperuser`
* 2- Prove your username, email and password (Recommended at least 8 characters with uppercase, lower case and numbers)
* 3- `python manage.py runserver`
* 4- Go to http://127.0.0.1:8000/admin
* 5- Insert your email and password
